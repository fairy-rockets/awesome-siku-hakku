# 四苦八苦なう

四苦八苦なうはあなたの苦しみをいい感じにTwitterでシェア―できちゃう革新的イノベーティブウェブサービスです。

画面上に過去2000年の知見から分類された典型的な苦しみがリストアップされていて、それを適当にクリックするだけで仲間にあなたの苦しみをふわっとした炎上無縁な感じで伝えることができます。

[早速苦しみをつぶやく](https://ledyba.github.io/awesome-siku-hakku/)
