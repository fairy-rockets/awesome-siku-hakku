# 四苦八苦なう

四苦八苦なうはあなたの苦しみをいい感じにTwitterでシェア―できちゃう革新的イノベーティブウェブサービスです。

画面をクリックして過去2000年の知見から分類された典型的な苦しみがリストアップされていて、それを適当にクリックするだけで仲間にあなたの苦しみをふわっとした感じで伝えることができます。

[お試しはこちら](http://ledyba.github.io/siku-hakku/)
